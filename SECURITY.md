# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **NOT** open a public issue.

Instead, please report it via privacy email to the maintainers (or check the repo owner's profile for contact details). We will review the issue and publish a fix as soon as possible.
