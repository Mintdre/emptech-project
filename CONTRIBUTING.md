# Contributing to Muse AI

Thank you for your interest in contributing to Muse AI! We welcome contributions from everyone.

## Getting Started

1.  **Fork** the repository.
2.  **Clone** your fork locally: `git clone https://codeberg.org/YOUR_USERNAME/emptech-project.git`
3.  **Install dependencies**: `npm install`
4.  **Create a `.env` file**: Copy `.env.example` and fill in the values.

## Development

*   Run the development server: `npm run dev` (requires Redis and Postgres running)
*   Lint code: `npm run lint`
*   Run tests: `npm test`

## Pull Request Process

1.  Ensure all tests pass.
2.  Update documentation if you change any functionality.
3.  Format your code using `npm run format`.
4.  Submit a Pull Request with a clear description of your changes.
